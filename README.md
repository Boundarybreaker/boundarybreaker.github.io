# About

My name is Meredith Espinosa, and I’m an independent programmer. I focus on developing systems and libraries for other developers to use. I am a frequent contributor to the Minecraft modding community, where I’ve helped write some of the core frameworks dozens of mods utilize.

# Projects

Here is a selection of projects I've worked on. More can be found on my [GitHub](https://github.com/boundarybreaker).

## Minecraft mods

- **[LibCD](https://github.com/cottonmc/libcd)** - Expansions to the data pack system in Minecraft 1.14 and above
- **[Trion](https://github.com/cottonmc/trion)** - Sci-fi combat mod for Minecraft 1.15 and above
- **[Universal Components](https://github.com/cottonmc/universalcomponents)** - Common, compatible storage for items, fluids, energy and data for Minecraft 1.15 and above
- **[Epicurean](https://github.com/cottonmc/epicurean)** - Advanced food system for Minecraft 1.14 and above
- **[Infra-Redstone](https://github.com/elytra/infraredstone)** 6-bit signal transfer system for Minecraft 1.12 and above
- **[Aquarius](https://github.com/boundarybreaker/aquarius)** - Additional ocean-related features for Minecraft 1.13 and above
- **[Cotton Scripting](https://github.com/cottonmc/cottonscripting)** - JSR223-based scripting for Minecraft 1.14 and above
- **[Bee Productive](https://github.com/AllOfFabric/BeeProductive)** - Traits and extended honey production for bees in Minecraft 1.15 and above
- **[Artis](https://github.com/AllOfFabric/Artis)** - Custom, data-driven crafting tables for Minecraft 1.15 and above
- **[Sandbox](https://github.com/sandboxpowered/sandboxapi)** - Abstracted library for modding Minecraft 1.14 and above
